# Plunk and Plunder - Quick Start Guide

## Getting Started

### Single Player
1. Launch the game
2. Click "Start Game" from main menu
3. Select unit → Right-click to move → Click "Pass Turn"
4. Destroy enemy shipyards to win!

### Multiplayer (Local Network)
1. **Host**: Click "Host Game" → Share your IP address
2. **Client**: Click "Join Game" → Enter host's IP → Click "Connect"
3. See `docs/MULTIPLAYER.md` for detailed setup

### Controls
- **Left Click**: Select unit/structure
- **Right Click**: Move selected unit
- **ESC**: Open menu
- **Pass Turn Button**: Submit orders and advance turn

## Basic Gameplay

### Resources
- **Gold**: Earned 100g per shipyard per turn
- Used for: Building ships (50g), deploying shipyards (100g), upgrades (60-100g)

### Ships
- Move up to 2 tiles per turn
- Can attack adjacent enemy ships (combat is automatic)
- Upgrade at friendly shipyards: Sails (speed), Cannons (attack), Max Life (health)

### Shipyards
- Deploy on harbor tiles (requires ship + 100g)
- Build new ships (50g, 3 turns to complete)
- Upgrade your ships

### Combat
- Automatic when units meet or pass each other
- Ships attack with cannons, reduce enemy health
- Destroyed ships are removed from the game
- Destroyed shipyards eliminate the player

### Victory
- Eliminate all enemy shipyards to win!
